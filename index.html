Основные команды

git clone - клонирует удаленный репозиторий

git add . - говорит гиту отслеживать все изменения в вашем локальном репозитории

git commit -m 'message' - гит сохраняет текущее состояние локального файла

git push - отправляет все изменения на удаленный репозиторий

git status - показывает статус на локальном репозитории

git pull - стягивает все оновления удаленного репозитория
